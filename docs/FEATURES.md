## AI-судья

**Система оценки:**
- Базовый шанс: 100%
- Критические проверки: опасные преступники → 0%
- Серьезные проблемы: нет размножения (-50%), заразные болезни (-70%), критическая фобия (-70%)
- Условие выживания не выполнено: -80%
- AI анализ: корректировка ±10%

**Финальное решение:** Случайное число < Шанс → Выжили ✅
